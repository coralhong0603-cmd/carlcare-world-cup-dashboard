# Carlcare Africa Facebook World Cup Dashboard

This folder is ready for GitHub Pages deployment.

## Entry Page

Open:

```text
index.html
```

## Deployment

Upload every file in this folder to a GitHub repository root, then enable GitHub Pages:

1. Repository `Settings`
2. `Pages`
3. Source: `Deploy from a branch`
4. Branch: `main`
5. Folder: `/root`

GitHub Pages will publish the site at:

```text
https://<github-username>.github.io/<repository-name>/
```

## Metric Scope

- Views, not Reach.
- Account Follows only.
